Meu treinamento é preparatório para a **HashiCorp Certified: Terraform Associate 003**. O material é 100% prático, focado em Microsoft Azure, com laboratórios, quizzes e um simulado completo. Além de preparar para a certificação, o treinamento é ideal para quem quer atuar com Terraform no mercado de trabalho.

### **Os temas cobrados no exame são**

* Understand infrastructure as code (IaC) concepts
* Understand Terraform's purpose (vs other IaC)
* Understand Terraform basics
* Use Terraform outside of core workflow
* Interact with Terraform modules
* Navigate Terraform workflow
* Implement and maintain state
* Read, generate, and modify configuration
* Understand Terraform Cloud

### **Sobre o exame**

O nível de dificuldade do exame se mantém constante, proporcionando uma avaliação justa e desafiadora. O formato também continua o mesmo, com uma duração de 1 hora e um total de 57 questões, o que considero suficiente para abranger os principais tópicos do Terraform.

Um aspecto que continua sendo um ponto negativo é a PSI, responsável pela supervisão do exame. O processo de verificação pré-exame ainda é um tanto quanto estressante, desta vez, chegando ao cúmulo de pedir a demonstração dos ouvidos para verificar a ausência de fones.

Apesar disso, o conteúdo do exame se mantém atualizado e alinhado com as necessidades do mercado. É gratificante ver que os esforços dedicados ao estudo se refletem na manutenção de um padrão elevado de conhecimento e habilidades.
